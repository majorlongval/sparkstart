### 3. Create your Project

```bash

```

**What happens next?**
1.  `projinit` will ask for a **GitHub Token**.
2.  It will **automatically open your browser** to the right GitHub page.
3.  Just scroll down, click **"Generate token"**, and paste it into the terminal.
